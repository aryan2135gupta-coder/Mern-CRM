# MERN CRM

A Customer Relationship Management project built with MongoDB, Express.js, React, and Node.js.

Current structure:

```text
backend/     Node.js, Express, MongoDB, Mongoose API
frontend/    React, Tailwind CSS, Vite app
```

Backend setup and API details are in `backend/README.md`.

## Run the Project

Start the backend:

```bash
cd backend
npm run dev
```

Start the frontend from the project root:

```bash
npm run dev
```

Frontend URL:

```text
http://localhost:5173
```
